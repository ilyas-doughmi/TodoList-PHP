# Task Manager

A very simple Task Manager web app built with **PHP**, **MySQL** and **Tailwind CSS**.  
It lets you quickly add tasks with a title and description, then displays all tasks stored in the database on the same page.

---

## Features

- Add a new task with a title and description via a simple form.
- Store tasks in a MySQL database using PHP (MySQLi).
- List all saved tasks under the form in a clean layout.
- Basic Tailwind CSS styling for a centered, responsive UI.

---

## Tech Stack

- **Backend:** PHP (MySQLi)
- **Database:** MySQL
- **Frontend:** HTML5, Tailwind CSS (CDN)
- **Server:** Any local server (XAMPP, WAMP, MAMP, Laragon, etc.)
